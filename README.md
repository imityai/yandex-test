# yandex-test

[Посмотреть](https://imityai.github.io/yandex-test/)

## Технологии

* HTML, SCSS, JS
* Gulp
* Webpack

### ТЗ

Сверстайте адаптивный лендинг по макету, используя стек html + css + чистый js (без библиотек и фреймворков).

* Придерживайтесь принципа Pixel Perfect.
* Избегайте дублирования текстового контента в мобильной и десктопной версиях.
* Добавьте бегущую строку.
* Кнопки на стартовом экране являются якорями и ведут к соответствующим блокам.
* Карусель с карточками участников должна быть зацикленной и меняться автоматически через каждые 4 секунды.
* Карусель с этапами не должна быть зацикленной и не должна автоматически менять слайды.
